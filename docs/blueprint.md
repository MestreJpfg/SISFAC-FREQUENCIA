# **App Name**: School Attendance Tracker

## Core Features:

- Excel Import: Import student data from an Excel file to create a Firestore database.
- Daily Attendance Marking: Mark daily attendance for students based on the imported data.
- Daily Absence Report: Generate a daily report of absent students.
- Monthly Absence Report: Generate a monthly report summarizing each student's absences.
- Data Storage: Store student data and attendance records in Firestore without requiring authentication for global access.

## Style Guidelines:

- Primary color: Soft pink (#F4B4C7) to create a welcoming and friendly interface, relating to school without being childish.
- Background color: Light pink (#F9E7EE), a very desaturated shade of the primary color.
- Accent color: Lavender (#D0B4F4) for interactive elements and highlights, creating a gentle contrast with the primary color.
- Body and headline font: 'PT Sans' (sans-serif) for clear and accessible text.
- Use simple, clear icons to represent different actions and data categories.
- Maintain a clean, uncluttered layout to ensure ease of use.
- Use subtle animations and transitions to enhance the user experience without being distracting.
